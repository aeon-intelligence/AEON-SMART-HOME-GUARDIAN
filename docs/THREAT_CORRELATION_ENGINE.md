# AEON MATRIX Threat Correlation Engine

## Mission

Analyze relationships between security events,
incidents, and indicators to improve threat intelligence.

## Capabilities

- Event correlation
- Pattern recognition
- Risk aggregation
- Incident relationship mapping
- Guardian Brain notification

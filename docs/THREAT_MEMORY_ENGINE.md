# AEON MATRIX Threat Memory Engine

## Mission

Create long-term security intelligence memory
for learning from historical incidents.

## Capabilities

- Store incident patterns
- Compare historical threats
- Maintain security knowledge
- Support risk prediction
- Provide learning signals to Guardian Brain

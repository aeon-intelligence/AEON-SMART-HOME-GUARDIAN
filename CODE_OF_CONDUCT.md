AEON Smart Home Guardian Code of Conduct

Our Commitment

The AEON Smart Home Guardian community is committed to providing a welcoming, respectful, inclusive, and professional environment for everyone.

Whether you are a maintainer, contributor, user, researcher, or community member, we expect all interactions to be constructive, collaborative, and respectful.

Our Standards

We encourage all community members to:

- Be respectful and professional.
- Be welcoming and inclusive.
- Communicate with patience and empathy.
- Provide constructive feedback.
- Respect different opinions and experiences.
- Collaborate in good faith.
- Help create a safe and positive community.

Unacceptable Behavior

The following behaviors are not acceptable:

- Harassment, discrimination, or intimidation.
- Personal attacks or abusive language.
- Hate speech or discriminatory comments.
- Threats or violent behavior.
- Sharing another person's private information without permission.
- Deliberately disrupting community discussions.
- Any conduct that creates an unsafe or hostile environment.

Scope

This Code of Conduct applies to all official AEON Smart Home Guardian community spaces, including:

- GitHub Repository
- GitHub Issues
- GitHub Pull Requests
- GitHub Discussions
- Project documentation
- Community events
- Any official communication channels managed by aeon-intelligence

Reporting

If you experience or witness behavior that violates this Code of Conduct, please report it by contacting:

Email: aeonmatrix.command@gmail.com

Reports will be handled respectfully and reviewed as promptly as possible.

Enforcement

Project maintainers are responsible for enforcing this Code of Conduct.

Depending on the severity of the violation, maintainers may:

- Provide guidance or a warning.
- Remove inappropriate content.
- Temporarily restrict participation.
- Permanently remove access to community spaces.

Our Values

- Respect
- Collaboration

- Responsible AI
- Security
- Professionalism
